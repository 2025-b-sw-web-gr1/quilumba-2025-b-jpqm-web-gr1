# Estándares del W3C

---

## 1️⃣ HTML5

### Descripción
HTML5 es el estándar principal para estructurar el contenido de la web. Define las etiquetas necesarias para crear elementos como texto, formularios, imágenes, videos y aplicaciones, siendo la base de todas las páginas modernas.

### Justificación
Se destaca porque es el estándar fundamental que permite la existencia de cualquier sitio web, siendo el punto de partida para el desarrollo web actual.

---

## 2️⃣ CSS

### Descripción
CSS define los estilos visuales de una página web, como colores, tamaños, tipografías, distribución y animaciones. Permite separar el contenido del diseño, logrando desarrollos más profesionales y organizados.

### Justificación
Es esencial porque proporciona el diseño visual e identidad gráfica a los sitios; sin él, las páginas tendrían una apariencia básica y poco atractiva.

---

## 3️⃣ XML

### Descripción
XML es un estándar para almacenar y transportar datos de forma estructurada, flexible y legible tanto por personas como por máquinas. Es utilizado ampliamente en configuraciones y transferencia de información entre sistemas.

### Justificación
Es importante porque representó un gran avance al permitir el intercambio ordenado y universal de datos entre diferentes plataformas.

---

## 4️⃣ DOM (Document Object Model)

### Descripción
El DOM define cómo acceder, modificar y actualizar dinámicamente la estructura, contenido y estilos de una página mediante lenguajes como JavaScript, permitiendo la creación de sitios interactivos.

### Justificación
Destaca porque es la base de la interactividad moderna en la web, haciendo posible animaciones, actualizaciones en tiempo real y aplicaciones web dinámicas.

---

## 5️⃣ WCAG (Web Content Accessibility Guidelines)

### Descripción
WCAG es un estándar que establece pautas para garantizar que las páginas web sean accesibles para personas con discapacidades visuales, auditivas, motoras o cognitivas.

### Justificación
Es fundamental porque promueve una web inclusiva en la que todas las personas puedan acceder y utilizar los contenidos sin barreras.
