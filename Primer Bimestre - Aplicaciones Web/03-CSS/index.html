Formas de cargar estilos en una página web

1) Inline: atributo style

Este párrafo usa inline style con el atributo style.

Elemento que será estilado desde JS con element.style.

2) Interno: etiqueta <style>

Este párrafo usa estilos definidos en la etiqueta <style> del documento.

3) Externo: archivo CSS enlazado con <link>

Este párrafo usa clases definidas en css/external.css.

4) @import dentro de una hoja CSS

La hoja external.css contiene un @import que trae css/imported.css.

5) Carga dinámica creando un elemento <link> (JavaScript)

Este párrafo se estilizará cuando cargues la hoja dinámica.

6) Carga por fetch e inyección en un <style> (JavaScript)

Este párrafo se estilizará cuando hagas fetch e inyectes el CSS.

7) Shadow DOM (estilos encapsulados)

Abre las herramientas de desarrollador para ver cómo se añaden los elementos <link> o <style> dinámicamente.
