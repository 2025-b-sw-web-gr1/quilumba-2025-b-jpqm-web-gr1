¿Qué es HTML?

HTML es el lenguaje estándar para crear páginas web. Utiliza "etiquetas" para estructurar el contenido.

Estructura Básica de una Página HTML

<!DOCTYPE html>
<html>
<head>
  <title>Título de la página</title>
</head>
<body>
  <h1>Mi primer encabezado</h1>
  <p>Mi primer párrafo.</p>
</body>
</html>

<h2>Este es un encabezado mediano</h2>

Elementos HTML

Un elemento HTML normalmente consta de una etiqueta de apertura, contenido y una etiqueta de cierre. Algunos elementos no tienen contenido ni etiqueta de cierre (elementos vacíos).

<p>Este es un párrafo.</p>
<a href="https://www.w3schools.com">Este es un enlace</a>

Ejemplo: Encabezados HTML

Los encabezados HTML van desde <h1> (el más importante) hasta <h6> (el menos importante). Así se ven:

<h1>Encabezado 1</h1>
<h2>Encabezado 2</h2>
<h3>Encabezado 3</h3>
<h4>Encabezado 4</h4>
<h5>Encabezado 5</h5>
<h6>Encabezado 6</h6>

Elementos Vacíos

Algunos elementos, como <br> y <img>, no tienen contenido y no requieren una etiqueta de cierre.
